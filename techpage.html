Relative, absolute, and fixed positioning in CSS

To understand the position property, we first need to understand that every web page element is a block. These elements respect the attributes that are set, like width and height. There are five values in the position property that allow the positioning of those blocks: relative, absolute, fixed, static and sticky, each of them is useful and the choice of which one to use depends on the desired result.
